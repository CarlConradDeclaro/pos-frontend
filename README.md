
## Getting Started

### Prerequisites

This project requires the following dependencies:

* **Programming Language:** TypeScript
* **Package Manager:** Npm

---

## Installation

Build `pos-frontend` from the source and install dependencies:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/CarlConradDeclaro/pos-frontend](https://github.com/CarlConradDeclaro/pos-frontend)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd pos-frontend
    ```

3.  **Install the dependencies:**
    Using [npm](https://www.npmjs.com/):
    ```bash
    npm install
    ```

---

## Usage

Run the project with:

Using [npm](https://www.npmjs.com/):
```bash
npm run dev
